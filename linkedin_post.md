🎯 Système de Prédiction de Risque de Crédit - AI Project

Fort de ma double compétence en Finance et Génie Logiciel, et en tant qu'étudiant en IA, je partage mon projet de système intelligent de prédiction du risque de crédit.

 🔍 Comment ça marche ?
L'application analyse les caractéristiques d'un demandeur de crédit (âge, emploi, logement, comptes bancaires, montant et durée du crédit) et prédit si la demande sera classified comme "good" (approuvé) ou "bad" (refusé).

 🎨 Interface utilisateur :
 • Dashboard interactif avec graphiques dynamiques
 • Formulaire de saisie intuitif
 • Visualisation en temps réel du score de risque
• Historique des prédictions

🤖 Intelligence Artificielle :
Le modèle utilise un Decision Tree Classifier entraîné sur le German Credit Data. L'IA explique également les facteurs qui ont influencé chaque prédiction !

Technologies utilisées :
Analyse de données : Pandas, NumPy, Matplotlib, Seaborn, Scikit-learn
- Nettoyage et prétraitement des données
- Exploration et visualisation des données (EDA)
- Entraînement et évaluation du modèle

 Backend : Python, Flask, Gunicorn, Joblib
 - API RESTful avec Flask
 - Sérialisation du modèle avec Joblib
 
Frontend : React 19, Vite 7, Tailwind CSS
 - Interface moderne et responsive
 
 Déploiement : Vercel (Frontend) + Render (Backend)
